# Discord As Current Vessel

## Definition

Discord is the current vessel for the K2M product kernel. It is where the student experiences onboarding, daily rhythm, agent access, peer visibility, facilitator escalation, and artifact publication.

## Pattern

Discord matters because it gives the product a social and behavioral container:

- private DMs create a low-pressure interface for reflection
- weekly channels make progress visible
- role and channel gates turn time into structure
- commands make agents callable at the moment of need
- the facilitator dashboard keeps human oversight close to the learner

Discord is not the product kernel. It is the first working habitat for the kernel.

## Diagnostic Use

Ask:

- Which parts of the product only work because Discord provides identity, channels, DMs, roles, and presence?
- Which parts would survive if the vessel changed?
- Where is Discord creating friction that hides the kernel?

## Trap

The trap is treating Discord limitations as product limitations. Some gaps belong to the vessel; some expose deeper product truth.
