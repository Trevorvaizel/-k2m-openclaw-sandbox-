# Metric 4: EXPLANATION CLARITY

> Simply can you explain it?

## Definition
Simply can you explain it?

## Pattern
- Can you explain your field to a 6-year-old?
- Test: Record yourself explaining a core concept
- Score: Can a beginner understand it immediately?
- Novice: Cannot explain (confuses themselves)
- Intermediate: Technical explanations (jargon-heavy)
- Advanced: Simple explanations (clear analogies)
- Master: Intuitive explanations (feels obvious)
- Conscious mastery (Stage 5)
