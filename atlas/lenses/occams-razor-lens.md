# Occam's Razor Lens

> Prefer the simplest explanation that adequately accounts for the facts.

## Definition
Occam's razor is a parsimony lens. It does not declare that the simplest story is always true; it says that explanations with fewer moving parts deserve the default preference until reality forces added complexity.

## Pattern
- A messy event generates multiple explanations.
- Narrative appetite pushes toward dramatic or overfitted stories.
- The simplest adequate explanation gets checked first.
- Complexity is added only when the simple explanation cannot carry the facts.

## Source-Grounded Example
The transcript uses the medical rule "when you hear hoofbeats, think horses, not zebras" to show that simplicity is a probability filter and a resource-allocation discipline. It also warns that the razor fails when used lazily in edge territories or scams where the surface simplicity is itself the trap.

## Diagnostic/Use
Ask: "What is the simplest explanation that fits the evidence without inventing extra machinery?"

Use this lens for diagnosis, debugging, investigation, and any environment where story inflation is outrunning the actual data.

## Trap
Using simplicity as a blunt instrument. A simple explanation that ignores domain context, tail risk, or contradictory evidence is not parsimonious; it is just shallow.
