# Mental Models as Lenses

> Mental models are interpretive lenses: they clear perception before the lattice connects knowledge.

## Definition
Mental models as lenses are reusable ways of seeing a territory before acting in it. They do not replace the manifesto or the lattice; they determine what the user can perceive, which distinctions become visible, and which connections the lattice is allowed to make.

## Pattern
- The user meets raw territory.
- A lens makes the territory legible.
- A single lens leaves blind spots; a cross-disciplinary set reduces them.
- The lattice connects those lenses to other models, forces, diagnostics, and cases.
- Better lenses create better links; weak lenses create [[danger-lattice-overload]].

## Source-Grounded Example
The belief-operations transcript set explicitly frames *The Great Mental Models* as the lens layer that helps a user see reality clearly before applying the belief operations manifesto. Transcript 10 then sharpens the point: a latticework is useful because different models catch different trains on the same track, but the lattice stays healthy only while those models keep contact with the ground.

## Diagnostic/Use
Ask: "Is this transcript giving me a new concept to store, or a lens that changes how existing concepts should be seen?"

Use a lens note when the durable value is the frame of perception. Use a concept/tool note when the durable value is the thing being described or the procedure being applied.

## Trap
Treating every mental model mention as a new atlas note. The lens earns a note only when it changes how multiple existing concepts are interpreted.
