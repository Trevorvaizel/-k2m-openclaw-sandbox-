# Stage 6: The Domain-Native Cartographer

> *"The field is speaking in its own structure."*

## State
The domain-native cartographer can transfer the cartographer method without importing foreign causality. Exemplars are used as probes, not as molds.

## Visible Signs
- names forces that survive multiple provinces of the field
- distinguishes imported language that reveals structure from language that hides it
- keeps the title, body, and claimed scope aligned
- accepts critique from practitioners in different provinces without treating the draft as finished canon

## Transition Trigger
Ask: Does this map still diagnose the field if I strip away the borrowed wording and test it against a second province?

## Trap
Confidence in the map hardens into certainty about the territory.
