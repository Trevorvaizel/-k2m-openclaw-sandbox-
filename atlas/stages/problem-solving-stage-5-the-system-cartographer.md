# Stage 5: The System Cartographer

> "I can map the forces before I move."

## Description
At this stage, the practitioner can externalize loops, delays, incentives, trust gaps, and decision rights that others only feel vaguely. They can choose when to use statistics, stories, experiments, or reference classes because the structure of the problem is becoming legible.

## What Becomes Possible
- translating diffuse failure into a readable system map
- selecting representations that clarify rather than obscure
- naming the active interaction between frame, constraint, causality, and delay

## Blind Spot
Building models that impress observers more than they change outcomes.
