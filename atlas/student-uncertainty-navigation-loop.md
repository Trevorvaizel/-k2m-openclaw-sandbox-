# Student Uncertainty Navigation Loop

## Definition

The student uncertainty navigation loop is the product pattern that helps a learner move from unstructured anxiety into owned thinking under uncertainty.

## Pattern

In the current product, the loop is:

1. Name the live decision context.
2. Frame the real question.
3. Add context and perspective.
4. Challenge the assumptions.
5. Synthesize a stance.
6. Create a proof artifact.
7. Use the artifact to revise identity: "I can think with AI, decide with clarity, and own the result."

This loop matters because the Kenyan transition context gives students high-stakes uncertainty with weak feedback, little margin of safety, and a heavy self-verdict.

## Diagnostic Use

Ask:

- Where is the student in the loop?
- What invisible self-verdict is blocking the next move?
- Has the product given them enough safety to revise the belief?
- What proof object shows the change?

## Trap

The trap is reducing the loop to curriculum pacing. The weekly structure is useful only if it helps the student revise how they see themselves and their decision.
