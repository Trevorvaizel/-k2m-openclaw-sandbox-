# Hybrid Agent Facilitation Model

## Definition

The hybrid agent facilitation model is the operating choice to let KIRA and automation handle the repeatable 90 percent of cohort work while Trevor holds the human 10 percent that still depends on judgment, culture sensing, and escalation.

## Pattern

The model works by replacing individualized facilitator attention with structural attention:

- mechanical verification replaces human noticing
- self-assessment replaces facilitator interpretation as the first signal
- peer visibility replaces most one-to-one feedback
- weekly spot-checking and escalation handling preserve a human backstop

This changes the product's success metric. The aim is no longer to maximize handcrafted transformation per student. The aim is to maintain framework integrity at scale while still producing meaningful transformation.

## Diagnostic Use

Ask:

- Which parts of the cohort should be executed by structure rather than by human labor?
- Where does human facilitation still matter because the risk is cultural, emotional, or interpretive?
- Are we optimizing for individual polish or for repeatable system integrity?

## Trap

The trap is pretending the hybrid model is just cheaper facilitation. It is actually a different product logic with different accepted outcomes and failure modes.
