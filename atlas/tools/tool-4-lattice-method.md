# Tool 4: The Lattice Method

> Build cross-disciplinary connections that earn their place by improving perception, judgment, or transfer.

## Purpose
Use the lattice method to keep knowledge from staying inert. A fact becomes useful when it is connected to other tested models that clarify where it applies, where it breaks, and what it predicts.

## Process
1. Learn the new concept or model.
2. Ask what existing models it strengthens, qualifies, or contradicts.
3. Test the connection against a real case, not just a verbal similarity.
4. Keep the link if it improves judgment or transfer.
5. Remove decorative links that add graph density without insight.

## Result
- New knowledge becomes easier to retain and reuse.
- Cross-domain transfer gets faster because models stop living in isolated silos.
- The user gains more ways to notice when a map no longer matches the territory.

## When To Use
Use this when you are learning across domains, extracting from source material, or trying to convert a pile of facts into a reusable judgment system.

## Failure Mode
Confusing more links with a better lattice. Untested or cosmetic links create overload, not understanding.
