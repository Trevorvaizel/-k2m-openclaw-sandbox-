# Tool 1: First Principles Deconstruction

> *Break everything down to "why." Understanding principles = 10x learning speed.*

## The Process
1. Take any technique/method/rule in your field
2. Ask: *"Why does this exist?"*
3. Ask: *"What problem does it solve?"*
4. Ask: *"What would happen if I removed it?"*
5. Ask: *"Is there a better way to solve that problem?"*

## The Result
You understand the **principle**, not just the technique.

## Force Connections
- Reveals [[asymmetric-leverage|Force 1]]: the principle is usually the leverage point
- Enables [[beginners-mind|Force 4]]: deconstructing "what everyone knows" reveals what's actually true
- Supports [[context-spectrum|Force 6]]: principles apply across contexts; techniques don't
