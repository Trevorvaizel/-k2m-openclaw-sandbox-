# Force Candidate Test

> A named force earns its place only when it survives causal, failure, transfer, and falsification pressure.

## Purpose
Use the force candidate test before promoting a recurring pattern into canon. It keeps beautiful language from outrunning the territory.

## Process
1. Check whether the proposed force is causal rather than merely descriptive.
2. Ask whether it explains failure, not only success.
3. Test whether it is native to the field rather than imported by habit.
4. See whether it transfers across more than one province or should be narrowed.
5. Ask whether it changes perception or action for a practitioner.
6. Ask what would falsify it.
7. Keep the name only if the compression remains useful after testing.

## Result
- weaker names stay patterns, tactics, or preferences
- stronger names become reusable force notes
- the canon gains fewer but more durable abstractions

## When To Use
Use this when a draft starts producing memorable phrases faster than it produces tested explanations.

## Failure Mode
Treating a good phrase as proof because it makes the draft feel coherent.
