# Practice 1: THE MORNING INQUIRY

> Start Every Day at Layer 4

## Purpose
Start Every Day at Layer 4

## Process
- Before starting work, ask: "What is the 20% that drives 80% today?"
- Identify the 1-3 leverage points for the day
- Ruthlessly eliminate everything else
- Work ONLY on those leverage points

## Result
- Forces attention to Layer 4 (invisible layer)
- Builds asymmetric leverage muscle memory
- Creates daily time compression
- Did I identify leverage points? (Yes/No)
- Did I work ONLY on leverage points? (Yes/No)
- If No, what distracted me? (Pattern awareness)
