# Galileo Ship Perspective Case

> Inside a system, the local view can be true and still incomplete.

## Definition
The Galileo ship case is a perspective case. A scientist inside a moving ship sees a ball drop straight down; an outside observer sees the ball moving both vertically and horizontally.

## Pattern
- A participant sees locally true evidence.
- The local frame hides a larger system movement.
- An outside observer sees the missing dimension.
- Better judgment requires both local experience and external perspective.

## Source-Grounded Example
The transcript uses the sealed-cabin scientist and the outside fish observer to show why participants inside a system need external perspective.

## Diagnostic/Use
Ask: "What motion am I inside that an outside observer would see?"

Use this case when a belief map is too participant-bound.

## Trap
Assuming outside observers are automatically right. The case argues for frame expansion, not blind deference to outsiders.
