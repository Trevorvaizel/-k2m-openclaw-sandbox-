# Linda Conjunction Fallacy Case

> A vivid narrative can feel more true while being statistically less probable.

## Definition
The Linda case shows how narrative fit can override basic probability logic.

## Pattern
- A rich description creates a strong character story.
- A more detailed option feels more plausible because it matches the story.
- The broader set is forgotten.
- Probability loses to narrative coherence.

## Source-Grounded Example
The transcript uses the Linda bank-teller example to show why "bank teller and activist" feels right while still being less probable than the simpler category "bank teller."

## Diagnostic/Use
Ask: "Am I choosing the better story or the more probable set?"

Use this case when a detailed explanation feels irresistible and you need to recheck the underlying probability structure.

## Trap
Thinking the case is only about formal logic. The same bias drives bad hiring reads, conspiracy thinking, and false malice attribution.
