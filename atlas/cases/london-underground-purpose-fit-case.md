# London Underground Purpose-Fit Case

> A map is accurate for the job it was built to do, not for every possible use of the territory.

## Definition
The London Underground purpose-fit case shows that map quality depends on intended use. A map can distort geography and still be excellent when the relevant structure is station connectivity.

## Pattern
- A map preserves one structure of the territory.
- The preserved structure makes one job easier.
- The omitted structure makes another job harder or dangerous.
- The user mistakes purpose-fit for universal truth.

## Source-Grounded Example
The transcript uses Harry Beck's Underground map to show structural similarity without geographic fidelity. It works for passengers transferring between lines, but it can mislead someone judging walking distance or operating a train.

## Diagnostic/Use
Ask: "What job was this map optimized for, and am I using it for that job?"

Use this case when a report, dashboard, MOC, or framework looks clean enough to feel universally reliable.

## Trap
Calling the map a lie because it fails a job it was not designed to perform. The sharper lesson is to match map, purpose, and territory.
