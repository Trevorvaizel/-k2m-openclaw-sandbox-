# Context-Collapse Force

> The same surface can carry incompatible worlds.

## Definition
The context-collapse force is the merging of incompatible roles, meanings, and trust conditions into one interface or stream, forcing the user to switch interpretive frames faster than they can metabolize.

## Pattern
- feeds and surfaces flatten unlike things into apparent equivalence
- commercial systems borrow trust from non-commercial contexts
- role confusion grows when the user is simultaneously learner, buyer, friend, voter, worker, or patient
- unlabeled context borrowing is a hidden source of manipulation

## Source-Grounded Example
The manifesto describes feeds where grief, comedy, politics, shopping, schoolwork, and AI explanation coexist, and notes how commerce, health, politics, and entertainment can borrow each other's authority or intimacy.

## Diagnostic / Use
Ask:
- Which contexts are being collapsed?
- Which context lends trust to another?
- Which role is the user in right now?
- Does the system help the user switch frames?
- Is commerce borrowing intimacy, authority, urgency, or vulnerability?

## Trap or Limitation
Assuming mixture itself is the failure. The problem is not combination alone; it is unlabeled or exploitative borrowing across contexts.
