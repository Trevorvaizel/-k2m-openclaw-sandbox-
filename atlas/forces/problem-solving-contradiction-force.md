# Contradiction Force

> Breakthroughs often come from refusing the false choice.

## Definition
The contradiction force is the move of questioning whether a tradeoff is truly structural or only appears inevitable inside the current representation of the problem.

## Pattern
- mediocre problem solving accepts inherited either-or constraints
- some tradeoffs dissolve when the boundary changes
- contradictions often survive because the optimized variable is wrong
- inventive progress frequently begins by challenging the inevitability of the tradeoff

## Source-Grounded Example
The manifesto uses pairs like faster or safer, cheaper or better, stronger or lighter to show how assumed contradictions can be artifacts of the current frame.

## Diagnostic / Use
Ask:
- Which assumption makes this tradeoff look inevitable?
- What changes if the system boundary changes?
- Are we optimizing the right variable?
- Is the contradiction real or merely inherited?

## Trap or Limitation
Pretending every tradeoff is false. Some constraints are real; the work is to test which is which.
