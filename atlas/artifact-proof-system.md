# Artifact Proof System

## Definition

The artifact proof system is the part of the product that turns invisible thinking change into a visible proof object for the student, peers, parents, and future institutions.

## Pattern

In the current playbook, the proof system includes:

- a structured thinking artifact built from framing, exploration, challenge, and synthesis
- showcase formats that make proof public without turning it into ranking
- metrics and verification logic that distinguish genuine shift from empty performance
- parent-report and graduation-card translations that carry the shift into other audiences
- privacy choices that let students decide whether proof is public, anonymous, or held privately with Trevor

This matters because the product is not finished when the student has a better conversation with AI. It is finished when the shift becomes legible.

## Diagnostic Use

Ask:

- What concrete object proves the learner changed?
- How does the product protect authenticity while still making proof shareable?
- Which stakeholders need a translated version of the same proof?

## Trap

The trap is treating the artifact as content output. Its real role is evidence of identity and reasoning change.
