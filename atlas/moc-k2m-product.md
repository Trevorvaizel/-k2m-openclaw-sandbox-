# MOC: K2M Product

> Use this map to connect the current Discord product to the deeper K2M product kernel and the wider atlas.

## How To Use This Map

Start here when an agent needs to answer what K2M is building, what exists today, or how the product connects to the manifestos. Treat Discord as the current vessel. Treat the product kernel as the deeper invariant.

For stepwise product-doc ingestion status, use [[k2m-product-docs-intake-ledger]].

## What This Map Reveals

K2M is not just an 8-week Discord cohort. The current product is a working vessel for a clarity engine: making invisible belief, identity, attention, and decision structures visible enough for a learner to revise them.
